# Sorting Algorithms

Algorithms:
  - BubbleSort
  - SelectionSort
  - InsertionSort
  - MergeSort
  - QuickSort
  - HeapSort
  - BucketSort
  - RadixSort

## Runtime Complexity

| Algorithms             |Best Case | Worst Case |
|------------------------|:--------:|:----------:|
| BubbleSort             | O(n^2)   | O(n^2)     |
| SelectionSort          | O(n^2)   | O(n^2)     |
| InsertionSort          | O(n)     | O(n^2)     |
| MergeSort              | O(?)     | O(nlog(n)) |
| QuickSort              | O(?)     | O(?)       |
| HeapSort               | O(?)     | O(?)       |
| BucketSort             | O(?)     | O(?)       |
| RadixSort              | O(?)     | O(?)       |
